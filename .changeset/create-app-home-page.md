Added a pre-configured Home page to the default create-app template. New Backstage instances now include the home plugin with a customizable widget grid, search bar, toolkit with configurable links and icons, world clocks, starred entities, and a Getting Started card with onboarding tips. The sidebar includes a Home link and the app-config includes instructions for switching the default landing page from Catalog to Home.
